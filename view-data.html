The Google Sheets edit link by itself is not a write API, so I could not safely connect the app directly to it from this codebase alone. If you publish one Google Apps Script Web App for that sheet and save that single URL on each device, all devices will upload into the same spreadsheet. Until then, each device can still download its local data as CSV for manual upload.
